# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, be constructive, assume good faith. Harassment, personal attacks, and
gatekeeping are not welcome here.

To report an incident, open a private report via GitHub or contact the maintainer. Reports are
handled confidentially.
